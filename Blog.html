Answer to the question number - 01

1. var and let create variables that can be reassigned another value.

2. const creates "constant" variables that cannot be reassigned another value.

3. let and const variable is block scope.But var variable is functional scope.

4. var can be updated and re-declared into the scope.But let and const cannot.

Answer to the question number - 02

1.Undefined :means a variable has been declared but has yet not been assigned a value.

2.NULL :Null is an assignment value. It can be assigned to a variable as a representation of no value.

3. Undefined is an ECMAScript1 (ES1) feature,NULL is a Primitive value in Javascript
